<style></style> = html에서 css언어를 사용할때 사용
{} = css 태그
바꾸길 원하는 태그 뒤에 {color:색상;} = 폰트 색상 변경 (ex:a{color:색상;})
<tag style=""> = html 태그 안에서 css언어를 사용할때 
text-decoration: none; = 텍스트의 밑줄을 삭제하고 싶을때
text-decoration: underline; = 밑줄을 넣고 싶을때
font-size: = 텍스트 크기 조절 ( %,px 등)
text-align: = 텍스트 정렬 (center,left,right 등)
border: = 테두리 
solid = 색
padding: = 테두리와 텍스트 간격
margin: = 테두리와 테두리 사이의 간격
display:inlin; = 간격 지우기 (display:block; = 간격 만들기) (display:none; = 안보이게)
list-style:none; = ul이나ol 사용시 나오는 기호나 숫자 제거 
mix max = 최소 최대
@media = 반응형 디자인
link rel="" = 외부 리소스 연결 (ex 스타일 코드:link rel="stylesheet" href="style.css")
